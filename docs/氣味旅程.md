# scent-journey.html — 煖意‧氣味旅程

> 展示會員歷月選香紀錄的時間軸頁面，透過 `userid` URL 參數識別會員，從 GAS 後端取回歷史資料並渲染。

---

## 頁面職責

- 以垂直時間軸呈現每月的選香記憶
- 標示季禮月（3月、6月、9月、12月）
- 本月進行中的節點以高亮樣式區隔
- 後端尚未串接時，自動回退至 Mock 設計稿資料

---

## URL 參數

| 參數 | 必填 | 說明 |
|---|---|---|
| `userid` | ✅ | LINE UID，由 `index.html` 跳轉時帶入 |

```js
const userId = new URLSearchParams(location.search).get('userid');
```

---

## API 呼叫

與 `index.html` 共用同一個 GAS 後端：

```
GET ${API_URL}?userid=<userId>
```

期望後端在 `isBound = true` 時，回應中包含 `history` 陣列：

```json
{
  "isBound": true,
  "history": [
    { "year": 2026, "month": 3, "monthIndex": 3, "perfumeName": "朝光", "isCurrent": true },
    { "year": 2026, "month": 2, "monthIndex": 2, "perfumeName": "夜和", "isCurrent": false }
  ]
}
```

| 欄位 | 型別 | 說明 |
|---|---|---|
| `year` | number | 西元年 |
| `month` | number | 月份（1–12） |
| `monthIndex` | number | 訂閱第幾個月（1 起算） |
| `perfumeName` | string \| null | 中文香水名；`null` 表示尚未選香 |
| `isCurrent` | boolean | 是否為本月 |

## 函式說明

### `buildItems(result)`

```js
function buildItems(result)
```

- 後端有回傳 `history` 陣列 → 使用真實資料
- 否則 → 使用 `MOCK_HISTORY`（三筆設計稿預覽）

### `renderItem(item)`

```js
function renderItem(item)
```

依 `item` 的欄位動態組出 HTML：

1. **`.history-item`** 包裝容器 + 時間軸圓點（本月為 `active`）
2. **`.history-date`** 顯示 `2026.03` 格式日期 + 第 N 個月標籤
3. **季禮卡片 `.milestone-gift`**（若 `month` 為 3/6/9/12 月）
4. **選香卡片 `.memory-card`**（有 `perfumeName` → 顯示香氣描述，無則顯示「整理中」）

### `loadJourney()`

```js
async function loadJourney()
```

主流程：
1. 從 URL 取 `userid`，無則顯示錯誤
2. `GET API_URL?userid=...`
3. `isBound = false` → 顯示「查無訂閱記錄」
4. 成功 → 呼叫 `buildItems` + `renderItem`，渲染至 `#timeline`

## 資料常數

### `SCENT_STYLES`

```js
const SCENT_STYLES = {
  '夜和': { barColor: '#D6CADD', desc: '...' },
  '朝光': { barColor: '#D0E5D2', desc: '...' },
  '暖居': { barColor: '#F4D5A6', desc: '...' },
};
```

定義各香水的左側色條顏色（`barColor`）與情境描述文字（`desc`）。

### `SEASON_GIFTS`

```js
const SEASON_GIFTS = {
  3:  { icon: '🌸', label: '春季專屬禮贈', barColor: '#FFB7C5' },
  6:  { icon: '🍧', label: '夏季專屬禮贈', barColor: '#AEE3E8' },
  9:  { icon: '🍂', label: '秋季專屬禮贈', barColor: '#E8C49A' },
  12: { icon: '❄️', label: '冬季專屬禮贈', barColor: '#C5D5E8' },
};
```

定義季禮月的圖示、標題、色條顏色。

## UI 元件

| 元件 | 說明 |
|---|---|
| `.nav-bar` | sticky 頂部導覽，含返回按鈕 `history.back()` |
| `.hero-section` | 頁面標題區 |
| `.timeline-container` | 垂直虛線時間軸容器 |
| `.history-item` | 單一月份時間軸節點 |
| `.timeline-dot.active` | 本月進行中，金色光暈 |
| `.milestone-gift` | 季禮卡片，含色條 + 圖示 + 說明 |
| `.memory-card` | 選香記憶卡，含左側色條 + 香水名 + 描述 |
| `.memory-card.unknown` | 尚未選香，`opacity: 0.45` 半透明 |
| `#endMark` | 頁尾裝飾符號 `❦`，資料載入完成後顯示 |
