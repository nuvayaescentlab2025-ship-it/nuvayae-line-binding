# index.html — 主頁（訂閱綁定系統）

> 煖意（Nuvayae）LIFF 應用程式的入口頁面，負責 LINE 登入、會員狀態確認、訂單綁定，以及會員儀表板。

---

## 頁面職責

| 職責 | 說明 |
|---|---|
| LIFF 初始化 | 載入 LINE LIFF SDK，確認使用者已登入 |
| 訂閱狀態查詢 | 向 GAS 後端 GET 請求，判斷是否已綁定 |
| 訂單綁定表單 | 未綁定者填入訂單編號 + 手機後 POST 綁定 |
| 會員儀表板 | 顯示訂閱進度卡、香氛年曆、選香轉盤、服務連結 |

---

| 常數 | 值 |
|---|---|
| `API_URL` | Google Apps Script 部署 URL |
| LIFF ID | `2009177086-5AGq54Qb` |

## UI 狀態

三個互斥 View，透過 `.hidden` CSS class 切換：

```
#loadingView   →  初始載入中
#unboundView   →  未綁定：顯示綁定表單
#boundView     →  已綁定：顯示會員儀表板
```

## 函式說明

### `initLiff()`

```js
async function initLiff()
```

- 呼叫 `liff.init({ liffId })`
- 若未登入則立即 `liff.login()`
- 登入後呼叫 `checkSubscriptionStatus()`

---

### `checkSubscriptionStatus()`

```js
async function checkSubscriptionStatus()
```

- 取得 LINE profile (`liff.getProfile()`)
- `GET ${API_URL}?userid=<lineUserId>`
- 後端回應結構：

```json
{
  "isBound": true,
  "duration": "3個月",
  "months": 3,
  "subType": "month" | "year",
  "currentPerfume": "夜和",
  "lastPerfume": "暖居",
  "deadline": "2026/04/06",
  "isLocked": false,
  "nextOrderDate": "2026/04/10"
}
```

- `isBound = true`：顯示 `#boundView`，依 `subType` 分流：
  - `year`：呼叫 `renderCalendar(months)`，移除擴香儀按鈕
  - `month`：顯示進度卡（`progress-card`）
- `isBound = false`：顯示 `#unboundView`

---

### `submitForm()`

```js
async function submitForm()
```

- 驗證 `orderId`、`mobile` 不為空
- `POST ${API_URL}` 傳送 `orderId, mobile, userid, displayName, pictureUrl`
- 成功（status 200）→ SweetAlert2 成功彈窗 → `location.reload()`
- 失敗 → 顯示錯誤訊息，按鈕解除 disabled

---

### `updateScentSelection(perfumeName)`

```js
function updateScentSelection(perfumeName)
```

- 根據香水名稱（中文）更新選香卡片按鈕狀態
- 已選定的卡片：`btn.classList.add('selected')`，文字改為「已選定」
- 其他卡片：移除 `selected`，文字改為「一鍵切換」
- 同步更新 `#reservedPerfumeName`

---

### `applyLockState(locked, nextOrderDate)`

```js
function applyLockState(locked, nextOrderDate)
```

- `locked = true`：所有 `.scent-card-btn` 設為 disabled，顯示 `#scentLockNotice`
- `locked = false`：隱藏 lock notice

---

### `renderCalendar(months)`

```js
function renderCalendar(months)
```

- 僅於 `subType === 'year'` 時呼叫
- 依目前月份 + 已訂閱月數，計算 12 個月節點的狀態：`past` / `current` / `seasonal` / `start-month`
- 季禮月固定為曆月：3月🌸、6月🍧、9月🍂、12月❄️
- 渲染至 `#calTimeline`，並顯示 `#yearCalendar`

## 選香轉盤（Carousel IIFE）

```js
(function () { ... })();
```

| 功能 | 說明 |
|---|---|
| 換頁 | 上一頁 / 下一頁按鈕、圓點指示器、Touch 滑動（≥40px 判定） |
| 選香 API | 點擊 `.scent-card-btn` 後 `POST action=select_perfume` |
| UI 更新 | 成功後呼叫 `updateScentSelection()`，更新已選定按鈕狀態 |

選香 POST 欄位：

```
action=select_perfume
userid=<LINE UID>
perfumeName=<中文香水名>
displayName=<LINE 顯示名>
```
