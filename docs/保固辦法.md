## 保固辦法（`.warranty-card`）

四行結構：

| 欄位 | 內容摘要 |
|---|---|
| 保固期限 | 每期配發設備，於對應訂閱期間內全程保固 |
| 保固範圍 | 機身馬達氣流問題免費換機；玻璃物流破損免費換新（皆免運） |
| 不在範圍 | 人為損壞、非純精油、外力碰撞、長期未清潔。玻璃人為損壞：基本款 $680、鬱金香款 $780 |
| 申請方法 | 拍攝故障影片，透過 LINE 客服提供姓名與影片 |
