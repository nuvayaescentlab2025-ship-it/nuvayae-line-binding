# fragrance-advisor.html — 專屬選香顧問

> 以聊天介面呈現的 AI 選香諮詢頁，透過 Supabase Edge Function 呼叫 AI 模型（Gemini），引導使用者填入六個需求槽位後給出香氣推薦。

---

## 頁面職責

- LIFF 登入取得 LINE UID（失敗時回退至 localStorage UUID）
- 渲染 AI 顧問「Honoka」與使用者的對話泡泡
- 追蹤六個需求槽位的填充進度
- 當 AI 判斷諮詢完成（`status: "completed"`），隱藏輸入框並顯示重啟按鈕

---

## API

詳細規格見 `docs/選香api.md`。

| 項目 | 值 |
|---|---|
| Endpoint | `POST https://kvqkjtyhbbcivwlexgcn.supabase.co/functions/v1/fragrance-advisor` |
| Content-Type | `application/json` |
| 驗證 | 無（不需要 JWT） |

Request Body：

```json
{
  "userId": "U1234...",
  "sessionId": "550e8400-...",
  "message": "我想找助眠香氛"
}
```

Response Body（status 200）：

```json
{
  "reply": "AI 顧問的回覆文字",
  "state": { "space": null, "mood": "助眠", "preferences": [], ... },
  "status": "active" | "completed"
}
```

| HTTP 狀態 | 說明 |
|---|---|
| 200 | 正常回應 |
| 409 | Session 已完成，拋出 `SESSION_COMPLETED` 錯誤 |

---

## 狀態變數

| 變數 | 型別 | 說明 |
|---|---|---|
| `userId` | string \| null | LINE UID 或 localStorage UUID |
| `sessionId` | string | 每次諮詢的 UUID（`crypto.randomUUID()`） |
| `chatStatus` | `'active'` \| `'completed'` | 對話是否已完成 |
| `isLoading` | boolean | 防止重複送出 |

---

## 六個需求槽位

| 槽位 key | 顯示名稱 | 說明 |
|---|---|---|
| `space` | 空間 | 使用空間（居家、辦公室⋯） |
| `mood` | 心情 | 情緒需求（放鬆、助眠⋯） |
| `preferences` | 香調 | 偏好香調陣列 |
| `dislikes` | 地雷 | 排斥香調陣列 |
| `occasion` | 場合 | 使用場合 |
| `health_conditions` | 健康 | 健康狀況（取代舊版 budget） |

槽位狀態以 DOM `#slot-<key>` 圓點表示，填充後加上 `.filled` class。

## 函式說明

### `init()`

```js
async function init()
```

1. `liff.init()`，登入後取得 `profile.userId`
2. 失敗時 fallback：讀寫 `localStorage.nuvayae_uid`
3. 呼叫 `appendAgentMessage()` 顯示歡迎訊息與快速選項

### `sendMessage(message)`

```js
async function sendMessage(message)
```

- POST 到 Supabase Edge Function
- HTTP 409 → `throw new Error('SESSION_COMPLETED')`
- 其他非 200 → 拋出錯誤訊息

### `handleSend(text)`

```js
async function handleSend(text)
```

主流程：
1. 防呆（空字串、`isLoading`、已完成）
2. 顯示使用者訊息（`appendUserMessage`）
3. 顯示打字動畫（`showTyping`）
4. 呼叫 `sendMessage`，取回後移除動畫、顯示 AI 回覆
5. 呼叫 `updateSlots(res.state)` 更新進度圓點
6. `status === 'completed'`：隱藏輸入區，顯示完成 Banner

### `appendAgentMessage(title, text, quickReplies?)`

- 在 `#chatTimeline` 末尾新增 AI 對話泡泡
- `title`：可選標題（`<h4>`）
- `text`：正文，使用 `white-space: pre-wrap`
- `quickReplies`：快速選項按鈕陣列，點擊後隱藏此組並呼叫 `handleSend`

### `appendUserMessage(text)`

- 在 `#chatTimeline` 末尾新增使用者對話泡泡

### `showTyping()` / `removeTyping()`

- 顯示 / 移除打字中動畫（三點跳動）

### `updateSlots(state)`

- 比對 `state` 各欄位是否有值，切換 `#slot-<key>` 圓點的 `.filled` class

### `escapeHtml(str)`

- 以 `div.textContent` 對使用者輸入做 HTML 跳脫，防止 XSS

### `scrollToBottom()`

- `requestAnimationFrame` 滾動 `#scrollArea` 至底部

## 事件綁定

| 元素 | 事件 | 動作 |
|---|---|---|
| `#sendBtn` | `click` | `handleSend(msgInput.value)` |
| `#msgInput` | `keydown Enter`（非 IME 組字中） | `handleSend(msgInput.value)` |
| `#backBtn` | `click` | `history.back()` 或跳轉 `./index.html` |
| `#restartBtn` | `click` | 重置所有狀態，產生新 `sessionId`，重顯歡迎訊息 |
