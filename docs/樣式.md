# style.css — 全域樣式

> 煖意（Nuvayae）LIFF 應用程式的共用樣式表，負責 `index.html` 主頁的版型與元件外觀。

---

## 品牌色

| 名稱 | Hex |
|---|---|
| 品牌金 | `#be9f74` |
| 深棕文字 | `#5d4a3e` |
| 淺灰輔助文字 | `#777` |

---

## 全域佈局

```css
body {
  background: url('background.jpg') no-repeat center center fixed;
  background-size: cover;
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}
```

- 背景為全版固定圖片（`background.jpg`）
- 內容卡片水平垂直置中

---

## #app 主卡片

```css
#app {
  background: rgba(255, 255, 255, 0.6);
  backdrop-filter: blur(10px);
  padding: 22px;
  border-radius: 16px;
  width: 80%;
  max-width: 320px;
}
```

- 半透明毛玻璃效果
- 最大寬度 320px，左右留白設計

## 元件樣式

### 輸入欄位

```css
input {
  width: 100%; padding: 12px; margin: 10px 0;
  border: 1px solid #eee; border-radius: 10px;
  background-color: rgba(255,255,255,0.8);
}
```

### 按鈕

```css
button {
  background-color: #be9f74; color: white;
  border-radius: 10px; font-weight: bold;
}
button:disabled { background-color: #ccc; }
button:active { transform: scale(0.96); }
```

### 連結群組 `.link-group a`

```css
.link-group a {
  background-color: #be9f74; color: white;
  border-radius: 10px; padding: 14px 16px;
  display: flex; flex-direction: column;
}
.link-title { font-size: 15px; font-weight: 700; }
.link-sub   { font-size: 11px; opacity: 0.88; }
```

## 進度卡 `.progress-card`

| 元素 | 說明 |
|---|---|
| `.progress-card` | 漸層半透明卡片，`fadeSlideIn` 動畫入場 |
| `.progress-fraction` | 月數大字，金屬漸層文字色 |
| `.progress-track` | 進度條軌道，圓角 10px |
| `.progress-fill` | 金屬填充，`transition: width 0.8s` |
| `.progress-headline` | 說明文字，深棕色 |

進度條填充漸層：

```css
background: linear-gradient(180deg,
  #f7e08a 0%, #d4a755 30%, #be9f74 60%, #9a7040 85%, #c8a870 100%
);
```

## 備註

`index.html` 內嵌 `<style>` 另外定義：
- 香氛年曆（`.cal-*`）樣式
- 選香轉盤（`.scent-*`）樣式
- 選香顧問 banner（`.agent-banner`）樣式
- 鎖定提示（`.scent-lock-notice`）樣式

這些元件樣式不在 `style.css` 中，詳見 `docs/index.md`。
